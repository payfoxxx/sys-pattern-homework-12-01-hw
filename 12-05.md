### Задание 2

```sql
EXPLAIN ANALYZE
SELECT DISTINCT CONCAT(c.last_name, ' ', c.first_name), sum(p.amount) over (partition by c.customer_id, f.title)
FROM payment p, rental r, customer c, inventory i, film f
where date(p.payment_date) = '2005-07-30' and p.payment_date = r.rental_date and r.customer_id = c.customer_id and i.inventory_id = r.inventory_id;
```
![До улучшения](https://github.com/payfoxxx/sys-pattern-homework-12-01-hw/blob/main/img/12-05/before_rework.png)

#### Анализируем запрос:
* Оконная функция sum over partition by + distinct убирают дубликаты, но занимает слишком много времени;
* Дата сравнивается приведенимем к date, происходит долго;

#### Исправляем запрос:
* Для удобства используем inner join, а не FROM и перечисление всех таблиц;
* Создаем индекс на поле payment_date, ускорит фильтрацию;
* Убираем distinct и делаем группировку по пользователю, и применяем корректно агрегатную функцию;

```sql
ALTER TABLE payment ADD INDEX idx_payment_date (payment_date);

EXPLAIN ANALYZE
SELECT CONCAT(c.last_name, ' ', c.first_name), sum(p.amount)
FROM payment p
INNER JOIN rental r ON p.rental_id = r.rental_id
INNER JOIN customer c ON r.customer_id = c.customer_id
INNER JOIN inventory i ON i.inventory_id = r.inventory_id
INNER JOIN film f ON f.film_id = i.film_id
where p.payment_date BETWEEN '2005-07-30 00:00:00' AND '2005-07-30 23:59:59'
group by r.customer_id;
```
