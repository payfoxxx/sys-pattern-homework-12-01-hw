# Домашнее задание к занятию «SQL. Часть 2» - `Матухно Игорь`

### Задание 1

``` sql
SELECT 
      CONCAT(staff.first_name, ' ', staff.last_name) AS `staff_name`
    , city.city
    , COUNT(*) AS `customer_count`
FROM store 
INNER JOIN customer AS c ON c.store_id = store.store_id
INNER JOIN staff ON store.manager_staff_id = staff.staff_id
INNER JOIN address ON store.address_id = address.address_id
INNER JOIN city ON address.city_id = city.city_id
GROUP BY store.store_id
HAVING customer_count > 300;
```

### Задание 2

```sql
SELECT COUNT(*) AS 'Counter'
FROM film
WHERE film.length > (SELECT AVG(film.length) AS 'average_length'
		FROM film
	);
```

### Задание 3

```sql
SELECT 
     YEAR(payment_date) AS 'year'
    , MONTH(payment_date) AS 'month'
    , SUM(amount) AS 'amount'
    , COUNT(rental.rental_id) AS 'rental_count'
FROM payment
INNER JOIN rental ON rental.rental_id = payment.rental_id
GROUP BY YEAR(payment_date), MONTH(payment_date)
ORDER BY amount DESC
LIMIT 1;
```

### Задание 4

```sql
SELECT
      s.staff_id
    , CONCAT(s.first_name, ' ', s.last_name) AS `staff_name`
    , COUNT(p.payment_id) AS `payment_count`
    , IF(COUNT(p.payment_id) > 8000, 'Да', 'Нет') AS `Премия`
FROM staff AS s
INNER JOIN payment AS p ON p.staff_id = s.staff_id
GROUP BY s.staff_id;
```

### Задание 5

```sql
SELECT f.title
FROM film AS f
LEFT JOIN inventory AS i ON f.film_id = i.film_id
LEFT JOIN rental AS r ON i.inventory_id = r.inventory_id
WHERE r.rental_id IS NULL;
```
